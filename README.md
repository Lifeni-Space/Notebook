# 笔记本

![GitHub](https://img.shields.io/github/license/Lifeni/notebook)

用来储存自己的笔记、文章与想法。


## 目录

重新调整后的目录结构如下：

| 目录            | 说明                                                     |
| --------------- | -------------------------------------------------------- |
| [存档](./存档/) | 存放 2018 年至 2022 年之间撰写和更新的文章               |
| [文章](./文章/) | 2023 年及之后撰写的文章，采用 [MDX](https://mdxjs.com/) 语法                   |
| [专题](./专题/) | 一些长期更新的内容，相比文章，展现方式和交互形式更加灵活 |
