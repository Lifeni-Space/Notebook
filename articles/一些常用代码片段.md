### 文字提示（Tooltip）

```css
span::after {
  content: attr(data-tooltip);
  position: absolute;
  left: 50%;
  top: calc(100% + 0.5rem);
  padding: 0.25rem 0.75rem;
  display: flex;
  border-radius: 0.25rem;
  color: #ffffff;
  background-color: #181818;
  white-space: nowrap;
  transform: translateX(-50%);
}
```

```html
<span data-tooltip="Hello"> CSS Tooltip </span>
```

别忘了把父元素的位置设置成 `position: relative;` 。
